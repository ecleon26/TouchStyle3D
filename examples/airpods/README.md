# AirPods Case — Application Example

This is the example shown on the cover figure of the TactStyle paper: an
AirPods case stylized with the surface microgeometry of a real stone roof.

## Files

| File | Description |
|---|---|
| `airpods.obj` | Base mesh (downloaded from Thingiverse). |
| `stone_heightfield.png` | Heightfield generated by TactStyle from a "round stone roof" texture image. |
| `airpods_styled_stone.obj` | Final stylized mesh, produced by displacing `airpods.obj` along its vertex normals using `stone_heightfield.png`. |

## Reproducing

From the repo root, with a fine-tuned checkpoint available:

```bash
# 1. Generate a heightfield from a texture image of your choice
python -m src.inference \
    --checkpoint  checkpoints/ckpt_vae_final.pth \
    --texture     /path/to/round_stone_roof.jpg \
    --output      examples/airpods/stone_heightfield.png

# 2. Apply that heightfield to the AirPods case
python -m src.apply_texture \
    --mesh    examples/airpods/airpods.obj \
    --height  examples/airpods/stone_heightfield.png \
    --output  examples/airpods/airpods_styled_stone.obj \
    --scale   0.05 \
    --remesh
```

Open the resulting `.obj` in Blender, MeshLab, or any 3D viewer to inspect the
displaced surface; export to STL for SLA / FDM 3D printing.
